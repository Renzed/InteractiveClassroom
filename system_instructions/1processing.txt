Users have generated ideas for marriage proposals. You will receive a JSON input with messages, a user_id and an ID for that message. For each message with a certain user_id, respond with the message_id, whether the message contained an idea for a marriage proposal, on a scale from 1-5 (with 5 being the best) how original the idea was given all ideas in the JSON input and on a scale from 1-5 (with 5 being the best) how the usefulness of the idea is.
